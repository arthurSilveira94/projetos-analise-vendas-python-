# Análise de Dados de Vendas com Python

Projeto de análise de dados utilizando Python, com foco em limpeza, transformação e geração de insights para apoio à tomada de decisão.

## Objetivo
Analisar dados de vendas para identificar padrões, produtos mais vendidos e desempenho por região.

## Dados
- Data da venda
- Produto
- Quantidade
- Preço
- Região

## Ferramentas
- Python
- Pandas
- Matplotlib

## Etapas
- Limpeza e tratamento de dados
- Criação de métricas (faturamento)
- Análise exploratória
- Visualização de dados

## Insights
- Identificação dos produtos mais lucrativos
- Regiões com maior volume de vendas
- Tendência de faturamento ao longo do tempo
